Antigravity Browser Control을 사용하여 아래 테스트를 실행해줘.

최근 10일 이내 변경사항을 분석해서 테스트 시나리오를 작성해주고,
Live View Mode 로 실행해서 브라우저화면에 테스트 진행상황을 볼 수 있게 해줘.
최종 결과는  실행 로그(Console Log)와 최종 성공/실패 리포트를 보여줘.

URL: localhost:5173

사전 작업: 로그인
시나리오 실행 전, 다음 순서로 로그인을 먼저 수행할 것:
1. URL 접속
2. 부서명은 "재정부", 역할은 "재정부", 사용자는 "fin001", 비밀번호는 "0000" 입력
3. 로그인 버튼 클릭 및 화면 로딩 대기

요청 사항으로는 
로그인 성공 후, 위 시나리오 ID에 정의된 테스트 케이스를 순차적으로 수행하고 화면 스크린샷을 찍어줘.
